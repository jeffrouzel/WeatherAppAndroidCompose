Description:

This project is an Android weather app that uses the OpenWeather API to display real-time weather data. It features two tabs: one for current weather showing location, temperature in Celsius, sunrise/sunset times, and dynamic weather icons, and another tab that lists weather data saved each time the app is opened.

TechStack:
Language: Kotlin
Platform: Android
UI Toolkit: Jetpack Compose
Material Design: Material Design 3 (Material3)
Architecture Pattern: MVVM (Model–View–ViewModel)
ViewModel (Compose integration): AndroidX Lifecycle ViewModel
Lifecycle-aware components: AndroidX Lifecycle Runtime
Dependency Injection: Dagger Hilt
Navigation: Jetpack Navigation (Compose Navigation)
HTTP Client: Retrofit
JSON Parsing: Gson
Network Logging: OkHttp Logging Interceptor
Concurrency: Kotlin Coroutines
